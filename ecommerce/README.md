E-Commerce Platform (Starter)
=================================

This folder contains a starter scaffold for a multi-vendor e-commerce platform.

What's included:
- SQL schema (3NF) in `backend/src/db/schema.sql`
- MongoDB product model in `backend/src/models/product_mongo.js`
- Minimal Express backend skeleton in `backend/src/index.js`
- Minimal Vite + React frontend in `frontend/`

Next steps:
- Install backend deps and run the server
- Run migrations to create SQL schema
- Seed MongoDB and Postgres with sample data

See the backend and frontend subfolders for usage.
